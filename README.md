# Mis Habitos

App web local para llevar tracking de habitos, metas personales y graficos de progreso.

## Comandos

```bash
npm install
npm run dev
```

La informacion se guarda en `localStorage` del navegador.

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub llamado `mishabitos`.
2. Sube este proyecto a GitHub.
3. En GitHub, abre `Settings > Pages`.
4. En `Build and deployment`, selecciona `GitHub Actions`.
5. Cada push a `main` publicara la app.

La app es instalable como PWA desde el navegador del celular con `Agregar a pantalla de inicio`.
